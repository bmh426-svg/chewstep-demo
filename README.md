# Chew_Step 데모 (씹기 발달 1분 체크 · 무료 베타)

이유식·유아식 전환기(6~24개월) 아이의 **씹기 발달**을 체크리스트 + 식사 영상으로 확인하고,
레벨별 맞춤 가이드 1페이지를 무료로 제공하는 **MVP 전 프로토타입(데이터 모집용)** 입니다.

- **단일 파일**: `index.html` 하나로 완결. 빌드 과정 없음(정적 사이트).
- **결제 없음**: 참여 → 씹기 Level + 가이드(보상) 제공.
- **영상은 저장하지 않음**: 브라우저 안에서 MediaPipe FaceLandmarker(CDN)로 씹기 지표만 추출해
  숫자(`video_metrics`)와 jawOpen/left/right 시계열(`video_series`)만 Supabase에 저장. 영상 픽셀은 기기를 떠나지 않음.
- **데이터 수집**: Supabase `demo_responses` 테이블에 **익명 INSERT**(동의 후). 개인 식별정보 미수집.

## 배포
정적 사이트라 Vercel에서 별도 설정 없이 배포됩니다(루트의 `index.html` 자동 서빙).

## 참고
- 클라이언트에 포함된 Supabase 키는 **anon(공개) 키**로, 노출돼도 무방합니다.
  RLS로 INSERT만 허용되고 조회는 막혀 있습니다.
- 씹기 지표는 겉으로 보이는 턱 움직임 기반 **추정치**이며 **의료 진단이 아닙니다.**
