원이 안심하고 도입하도록

아이의 영상은, 동의한 만큼만 다룹니다.

기관 촬영은 원본 영상을 편집·분석하기 때문에, 개인정보 보호를 설계의 가장 앞에 두었습니다. 원의 책임 부담을 줄이도록 동의는 보호자 본인이 직접 하고, 수신 방식도 보호자가 선택합니다.

① 보호자 직접 동의

원의 대리 동의가 아니라 보호자 본인이 직접 동의해야 촬영·분석이 시작됩니다. 동의하지 않은 원아는 제외됩니다.

② 수신 방식 3택

보호자가 식사 영상 / 분석 자료만 / 썸네일만 중에서 선택합니다. “분석 자료만”을 고르면 분석 후 원본은 즉시 파기됩니다.

③ 접근 권한 분리

교사는 담당 반만, 학부모는 자기 아이만 봅니다. 원본 영상은 지정된 편집·관리 권한에서만 접근되고 기록이 남습니다.

원본 영상 보관·파기 세부 정책과 동의서는 법무 검토를 거쳐 확정합니다. 파일럿은 이 검토가 완료된 범위에서 운영합니다.
